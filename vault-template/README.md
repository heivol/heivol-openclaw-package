# Your Vault

This directory is your personal knowledge base. The openclaw-package writes into it; you own it.

## Layout

- `diary/` — daily capture files (`YYYY-MM-DD.md`), one per day
- `entities/` — pages about people, projects, companies, tools
- `sources/` — ingested articles, documents, transcripts
- `meta/` — generated by the package
  - `signals/` — session metadata JSONL (one file per day)
  - `analysis/` — daily meta-harness reports
  - `proposals/` — suggested improvements when a pattern recurs
- `MEMORY.md` — index of durable facts (see note inside)

## Obsidian

Open this directory as an Obsidian vault for a browsable UI. No Obsidian plugins are required; everything is plain markdown + wiki-links.

## Privacy

The package never sends any content in this directory to a third party unless you configure `llm_cmd` in `~/.openclaw/user/config.yaml`. Even with `llm_cmd` set, only categorization prompts and the metadata-only meta-analysis call the LLM — your diary stays local otherwise.
